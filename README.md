## 📖 Tentang Proyek

**AndirH Store** adalah website toko online sederhana yang dibuat sebagai media pembelajaran pengembangan web. Website ini memiliki tampilan modern, responsif, dan mudah digunakan untuk menampilkan produk serta melakukan simulasi belanja.

---

## ✨ Fitur

- 🛍️ Menampilkan daftar produk
- 🔍 Fitur pencarian produk
- 🛒 Keranjang belanja
- ➕ Menambah & mengurangi jumlah produk
- 💰 Perhitungan total harga otomatis
- 📱 Responsive di Desktop & Mobile
- 🎨 Desain modern dan user-friendly

---

## 🛠️ Teknologi

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🚀 Cara Menjalankan

1. Clone repository

```bash
git clone https://github.com/andirayadihidayat-jpg/AndirH-Store.git
```

2. Masuk ke folder project

```bash
cd AndirH-Store
```

3. Jalankan dengan membuka file **index.html** di browser.
